# Video Reels Directory

Each project folder contains placeholder files for 6 video reels in **9:16 aspect ratio** (1080x1920 — Instagram Reels / TikTok / YouTube Shorts format).

## How to Replace

1. Drop your `.mp4` video files into the correct project folder
2. Name them `reel-1.mp4`, `reel-2.mp4`, ... `reel-6.mp4`
3. The website will automatically display them in the carousel

## Video Specifications
- **Aspect Ratio**: 9:16 (vertical)
- **Resolution**: 1080x1920 recommended
- **Format**: MP4 (H.264 codec)
- **Duration**: 15-60 seconds
- **File Size**: Keep under 10MB for web performance
